# GradientFlow

## Description
GradientFlow is a modern animated gradient background project built using pure HTML and CSS.  
It creates smooth, eye-catching color transitions using CSS keyframe animations.

---

## Features
- Smooth animated gradient background
- Pure CSS (no JavaScript)
- Modern UI look
- Fully responsive
- Easy to customize colors and speed

---

## Technologies Used
- HTML5
- CSS3
  - Linear gradients
  - Keyframe animations
